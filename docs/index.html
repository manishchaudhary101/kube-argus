KUBE-ARGUS

The Kubernetes dashboard you'd build if you were tired of switching between k9s, Grafana, and kubectl.
Live cluster state, streaming logs, just-in-time exec access, cost analysis, and AI-powered diagnostics — in a single binary with zero dependencies.

GitHub Artifact Hub
Install via Helm
helm install kube-argus oci://ghcr.io/manishchaudhary101/charts/kube-argus \ --set env.CLUSTER_NAME="my-cluster"

What you get

⚡

Real-time Cluster State

Nodes, pods, workloads, services, and events — auto-refreshing with zero manual reloads.

📜

Streaming Logs & Exec

Live pod logs via SSE with follow mode, previous container logs, and browser-based terminal via xterm.js.

🔧

Node Drain Wizard

Safe, guided drains with PDB awareness, eviction preview, and real-time progress streaming.

💰

Cost Analysis

Namespace cost allocation, spot instance advisor, and resource right-sizing recommendations.

🤖

AI Diagnostics

LLM-powered root cause analysis for troubled pods. Works with any OpenAI-compatible API.

📝

YAML Editor

View and edit raw YAML for 11 resource types directly from the dashboard with syntax highlighting.

🔗

Dependency Graph

Visual map of Deployments → Services → Ingresses → HPAs → Secrets with config drift detection.

🚨

Troubled Pods Screen

A dedicated, high-signal view of only broken pods — perfect for NOC teams and incident response.

🔑

Just-in-Time Exec Access

Zero-trust shell access — viewers request time-bound exec, admins approve/deny. Workload-scoped, auto-expiring, fully audited.

🔐

SSO & RBAC Built In

Google SSO, any OIDC provider (Okta, Auth0, Keycloak), or no auth. Role-based access out of the box.
